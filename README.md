# 凯撒密码可视化实现
本项目是基于Python语法实现的凯撒密码加密工具，支持大小写字母的加密，并增加了字母移位可视化功能。
# 凯撒密码原理
凯撒密码是一种经典的替换加密技术，通过将字母按照指定的偏移量进行移位实现加密（如 A+3=D，z+1=a），解密则是加密的逆过程。
本项目支持：
- 大小写字母的加密与解密
- 字母移位可视化（柱状图）
